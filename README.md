# TFGAppEstudiantesIntercambio

Mobile app developed in Android Studio (Kotlin) to help exchange students feel at home in their new city, with multiple user roles (super admin, city admin, and student), scalable structure for any city, an adaptable info section with local tips and essential resources, and a community-building activities section where students can propose, join, and manage events, among other things.

Built using Android SDK, Kotlin, Firebase authentication and database integration, ViewModel/LiveData architecture, and Gradle for project configuration.
